Eclipse Platform Java User Interface

This document lists the extension points that the Eclipse Platform Java User Interface makes available to tool developers. 

Extension Points

The following extension points can be used to extend various aspects of the UI, both visually and functionally:
  • org.eclipse.jdt.ui.classpathContainerPage
  • org.eclipse.jdt.ui.javaEditorTextHovers
  • org.eclipse.jdt.ui.javaElementFilters
