Application programming interfaces for interaction with the Eclipse Java User Interface.

Package Specification

The JavaUI provides access to the Java user interface components. It allows to programmatically open editors on Java elements and open package and type prompter dialogs. The package exposes constants for retrieving Java user interface parts from the workbench registry.

The JavaUI class gives access to the IWorkingCopyManager, it provides access to the working copies that are in use for Java compilation units.

The interfaces ITypeHierarchyViewPart and IPackagesViewPart define the programming interface to interact with the corresponding view parts.

The classes JavaElementContentProvider and JavaElementLabelProvider support presenting Java elements in the Eclipse UI.
